### Summary
- **Tags**: Basic building blocks of HTML (e.g., `<p>`, `<h1>`).
- **Elements**: The combination of tags and content (e.g., lists, tables).
- **Attributes**: Extra information about elements (e.g., `src`, `alt`, `href`).
